# makefile

### 📄 Description

File generator for NestJs projects.

### ⚙️ Requirements

node >= 12 and npm >= 6

### Installation

```bash
$ npm i -g @viviflowt/makefile
```

### Usage

```bash
$ makefile [options] <name>
```
