# @viviflowt/makefile

### _MakeFile CLI_ (Under Develpment)

### Description

File generator for NestJs projects.

### Requirements

node >= 12
npm >= 6

## Installation

```bash
$ npm i -g @viviflowt/makefile
```

### Usage

```bash
$ makef [options] <name>
```
